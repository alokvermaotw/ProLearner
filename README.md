# ProLearner
An extension for the chromium based browser to help you learning made easy

## feature
- splits the page in two vertical parts and takes 25% width right side for LLM and 75% width for web page
- In the LLM panel it will ask for the api key setup on which user can select the api key provider like openai or gemini and enter the api key once entered it will be stored on top inside modal view of setting icon (like in chrome) in llm panel. From where user can signup on the proLearner as well to store their browser history too. and before the setting icon add an icon for expand the LLM Panel to 50% width. and before that icon add an icon to close the LLM Panel.
- what ever the current active page show a summary for that page and on top of that there will be a prompt box to ask anything related to that page, on top of that there will be buttons to switch between different mode like chat, summarizer, flashcards, quiz, and notes
- chat mode is for the chatting with the LLM, In chat mode user can ask anything related to the current page and LLM will answer it
- summarizer mode is for the summarizing the current page in bullet points
- flashcards mode is for the generating flashcards based on the content of the current page and save the flashcards in the local storage of the browser. It will have sync option with the proLearner account using API key.
- quiz mode is for the taking the quiz from the LLM, it will generate the quiz based on the content of the current page and save the quiz in the local storage of the browser. It will have sync option with the proLearner account using API key.
- notes mode is for the taking the notes from the LLM, it will store notes in the local storage of the browser. It will have sync option with the proLearner account using API key.

## use case
- user can use this extension for their learning purpose.
